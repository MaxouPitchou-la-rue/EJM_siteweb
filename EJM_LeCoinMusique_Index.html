Présentation

Le Coin Musique est un site, créé par des étudiants en BTS SIO, qui permet aux utilisateurs de découvrir les différents types musicaux afin d’enrichir leur culture musicale.

La découverte se fait via des écoutes de différentes musiques avec une découverte auditive.

Histoire

Dans le cadre de notre formation, nous avons décidé de créer le site Le Coin Musique afin de permettre aux utilisateurs de découvrir différents types de musiques.
